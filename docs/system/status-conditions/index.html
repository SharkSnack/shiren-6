Status Conditions

Edit on GitHub

Overview

Status conditions are states that affect a character's behavior in various ways.
Most status conditions besides "Normal" can be categorized as positive, neutral, or negative.
Eating a Peach Bun or reverting from Morphed status cures all status conditions.
