Hidden route that unlocks when you've reached Beachside Pier once and visit Small Harbor twice?
The floor numbers are displayed as 1-7F, but the dungeon corresponds to 10-15F of Serpentcoil Island.
Blue stairs which let you return to regular Serpentcoil Island floors early can be found starting on 4F.
Once unlocked, this dungeon can be entered anytime.

Strategy

General

The main characteristic of this dungeon is that Behemoth Monsters can occur on all floors.
In addition, only monsters with swift action speed spawn between 1-4F, so retreating is ineffective.
This means you'll need a shield strong enough to withstand Karakuroid and Hell Reaper's attacks.
That said, Karakuroid has 20 attack and Hell Reaper has 12 attack, so the requirement isn't very high.
Otherwise, the only other monster that spawns is Bowboy between 5-7F, so it's not too bad.

Overall, this dungeon is mostly pointless, since shops don't spawn and you miss out on a blacksmith.
The only real benefit is that blue sacred items can occasionally be found on the ground.
