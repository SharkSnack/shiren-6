Road to the Hamlet

Edit on GitHub

Tutorial dungeon where advice about the basics of dungeon gameplay appear as you progress.
Items you pick up are kept after clearing the dungeon, so be sure to check every room on each floor.
It's probably safe to say the only way to fail it is if you intentionally collapse.
Once cleared, you can't return to this dungeon.
