Scrolls

Edit on GitHub

Overview

  • Read a scroll to cause an effect to occur.
    • Sanctuary Scroll should be placed on the ground instead.
    • Scroll of Removal should be thrown at a monster instead.

Summary

Warps you to a different room first if read in a hallway.
Scroll of HasteAll non-player characters in the room gain speed (1 stage).

Small chance to identify all items not inside pots.

Hands Full ScrollMakes you unable to pick up items while on the current floor.

Scroll of Removal300009000Throw this at a monster to make that monster family no longer spawn.
Effect lasts for the duration of the current adventure.
Blank Scroll Exclusive
Sanctuary Scroll--Place this on the ground to create a safe zone where adjacent
monsters can't use direct attacks against anyone standing on the scroll.
How to Unlock: Clear Table Mountain.
Destruction Scroll--Wipes out all monsters in the same room.
How to Unlock: Clear Table Mountain without using fast-travel,
storehouses, Storehouse Jar, or the storehouse NPC.

1 Power Up Scroll: x1.50 damage, additional +0.25 to multiplier per scroll use. (Max: x3.25 at 8 scrolls)

2 Scroll of Need:
Fully restore HP - When remaining HP is low.
Fully restore strength - When current strength is 1.
Fully replenish fullness - When fullness is currently at 0.
Paralyze enemies - When 2 or more enemies are adjacent.
Cure ailments - When afflicted by select negative status conditions.
Remove curse - When an equipped item is cursed.
Gain ~5000 Gitan - When you're carrying 0 Gitan.
